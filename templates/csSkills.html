  • Minimum Spanning Trees

    I developed two programs in Java that use Dijkstra's and Kruskal's algorithms to find minimum spanning trees between US cities. Through a GUI, users can select two cities and the MST is displayed graphically.

    I wrote a Java program that utilizes a Minimum Binary Heap to determine the character frequency of a file, simulating the underlying process of file compression. The program reads through the file and builds a binary heap corresponding to character frequency in the given file.

  • C++ Templates

    This C++ program creates a list class (TList) a few different ways. The first is with an underlying C linked list which I wrote previously, the second is templated with a private C++ list member so it can be used with any data object, and the third is also templated with a private C++ vector member so the [] operator can be used more efficiently.
  • Expression Tree Evaluator

    I wrote a Java program which uses a stack to evaluate expression trees. The program pushes integers on the stack, pops and evaluates them on an operand, and then pushes the result back on the stack. At the end of the expression, the stack is checked to ensure there is just one integer, the result, and pops said result and returns it to the user.
